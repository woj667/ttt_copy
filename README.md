# tictactoe

Tic-tac-toe (American English), noughts and crosses (British English) is a game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. 

### Prerequisites

What things you need to install the software and how to install them:

```
* Python3 interpreter
* tkinter library for python
```

### Installing

A step by step series installation guide:

```
1. Get & install Python (https://www.python.org/downloads/) for your system
```
```
2. Open terminal & enter a path of downloaded project
```
```
3. Using 'python3 main.py' command run the game
```
```
4. Enjoy! :)
```
### Troubleshooting

In case of missing tkinter (ModuleNotFoundError: No module named 'tkinter')

```
1. Use command apt-get install python3-tk to install missing library
```
```
2. Run the game
```
